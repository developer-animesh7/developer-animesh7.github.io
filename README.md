# Animesh Patra — Portfolio

Static, fast portfolio site (HTML/CSS/JS).

## Run locally

Option 1: VS Code Live Server extension
- Open `index.html` and click **Go Live**.

Option 2: Python simple server
- From this folder, run:
  - Windows PowerShell:
    - `cd "c:\My Portfolio website"; & "c:\My Portfolio website\.venv\Scripts\python.exe" -m http.server 5173`
- Open `http://localhost:5173`.

## Add your profile photo

- Put your photo at `assets/profile.jpg` (recommended square-ish crop).
- If it’s missing, the site falls back to `assets/profile-placeholder.svg`.

## Edit content

- Main content: `index.html`
- Styles: `styles.css`
- Small JS (theme + mailto form): `script.js`
